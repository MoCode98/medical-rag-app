## 🆕 Feature 7: Enhanced PDF Parsing

### What's New
**Module:** `src/enhanced_pdf_parser.py`

Extract **tables, figures, and references** from medical research PDFs automatically.

### Capabilities

#### 1. **Table Extraction**
- Detects and extracts tables from PDFs
- Converts to markdown format
- Finds table captions automatically
- Preserves table structure

#### 2. **Figure Detection**
- Locates figures/images in PDFs
- Extracts bounding box coordinates
- Finds figure captions (Fig. X, Figure X)
- Tracks figure locations

#### 3. **Reference Parsing**
- Extracts bibliography/references section
- Parses individual citations
- Extracts authors, years
- Numbers references automatically

### Usage

**Python API:**
```python
from src import EnhancedPDFParser

# Initialize parser
parser = EnhancedPDFParser()

# Parse single PDF with enhanced extraction
result = parser.parse_pdf_enhanced(pdf_path)

print(f"Tables: {len(result['tables'])}")
print(f"Figures: {len(result['figures'])}")
print(f"References: {len(result['references'])}")

# Access table data
for table in result['tables']:
    print(f"Table on page {table.page_number}")
    print(table.content)  # Markdown format
    if table.caption:
        print(f"Caption: {table.caption}")

# Access figure data
for figure in result['figures']:
    print(f"Figure on page {figure.page_number}")
    if figure.caption:
        print(f"Caption: {figure.caption}")

# Access references
for ref in result['references']:
    print(f"{ref.reference_number}. {ref.reference_text}")
    if ref.authors:
        print(f"   Authors: {ref.authors}")
    if ref.year:
        print(f"   Year: {ref.year}")
```

**Parse all PDFs:**
```python
# Enhanced parsing for all PDFs
results = parser.parse_all_enhanced(save_json=True)

# Results saved to: data/enhanced_parsing_results.json
```

## 🆕 Feature 8: Question Type Classification & Adaptive Retrieval

### What's New
**Module:** `src/question_classifier.py`

Automatically classifies questions by type and **adapts retrieval strategy** for optimal results.

### Question Types

| Type | Examples | Retrieval Strategy |
|------|----------|-------------------|
| **FACTUAL** | "What is stroke?", "Define hypertension" | top_k=3, precise, temp=0.1 |
| **COMPARISON** | "Compare X and Y", "Difference between" | top_k=8, multiple perspectives |
| **CAUSAL** | "Why does X?", "What causes Y?" | top_k=6, deep context, expansion |
| **PROCEDURAL** | "How to X?", "What steps?" | top_k=5, sequential, no rerank |
| **SUMMARY** | "Summarize X", "Overview of Y" | top_k=10, broad coverage |
| **NUMERICAL** | "How many?", "What percentage?" | top_k=3, precise, temp=0.0 |
| **EVIDENCE** | "What evidence?", "What studies show?" | top_k=8, multiple sources |
| **OPINION** | "What do experts recommend?" | top_k=6, guidelines |

### How It Works

**1. Classification:**
- Rule-based (fast, pattern matching)
- LLM-based (accurate, slower)

**2. Adaptive Parameters:**
Each question type gets optimal:
- `top_k`: Number of chunks to retrieve
- `use_hybrid`: Whether to use hybrid search
- `use_reranking`: Whether to re-rank
- `use_expansion`: Whether to expand query
- `temperature`: LLM generation temperature

### Usage

**Python API:**
```python
from src import QuestionClassifier, AdaptiveRetriever, ConversationalRAG

# Initialize classifier
classifier = QuestionClassifier(use_llm=False)  # or True for LLM

# Classify a question
qtype = classifier.classify("What is the sample size?")
print(f"Type: {qtype.value}")  # "NUMERICAL"

# Get optimal retrieval params
params = classifier.get_retrieval_params(qtype)
print(params)
# {
#   'top_k': 3,
#   'use_hybrid': True,
#   'use_reranking': True,
#   'temperature': 0.0
# }

# Adaptive retrieval
rag = ConversationalRAG()
adaptive = AdaptiveRetriever(rag, classifier)

# Query with adaptive strategy
response = adaptive.query_adaptive("Compare stroke types")
# Automatically uses comparison strategy (top_k=8, etc.)
```

**Manual Testing:**
```python
# Test classification
classifier = QuestionClassifier()

questions = [
    "What is stroke?",  # FACTUAL
    "Compare ischemic and hemorrhagic stroke",  # COMPARISON
    "Why does hypertension cause stroke?",  # CAUSAL
    "How to prevent stroke?",  # PROCEDURAL
    "Summarize stroke risk factors",  # SUMMARY
]

for q in questions:
    qtype = classifier.classify(q)
    print(f"{q} → {qtype.value}")
```

## 🔄 Integration

### Combined Usage

Use all features together:

```python
from src import (
    EnhancedPDFParser,
    ConversationalRAG,
    AdaptiveRetriever,
    QuestionClassifier
)

# 1. Parse PDFs with enhanced extraction
parser = EnhancedPDFParser()

print(f"Extracted: {len(results['tables'])} tables, "
      f"{len(results['figures'])} figures, "
      f"{len(results['references'])} references")

# 2. Set up adaptive RAG
rag = ConversationalRAG(
    use_hybrid_search=True,
    use_reranking=True
)

classifier = QuestionClassifier()
adaptive = AdaptiveRetriever(rag, classifier)

# 3. Query with adaptive retrieval
response = adaptive.query_adaptive(
    "Compare the treatment protocols in the studies"
)

# Question automatically classified as COMPARISON
# Retrieval strategy adapted accordingly
```
