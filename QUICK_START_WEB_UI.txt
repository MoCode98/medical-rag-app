│ 2. (OPTIONAL) ADD PDFs FOR AUTO-INGESTION                                   │

└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ FEATURES                                                                      │
└──────────────────────────────────────────────────────────────────────────────┘

  🚀 AUTO-INGESTION (NEW!)
     • Automatically ingests PDFs from pdfs/ folder on startup
     • Only processes new files (uses progress tracking)
     • Runs in background - UI immediately accessible
     • Non-blocking status banner shows progress
     • Queries available even during ingestion

  📄 INGEST TAB
     • Drag-and-drop PDF upload
     • Real-time progress with WebSocket
     • Terminal-style logs
     • Database reset option

  💬 QUERY TAB
     • Chat interface with history
     • Configure model, top-k, temperature
     • Markdown-formatted answers
     • Source citations with scores
     • Always available (even with empty database)

  📊 METRICS TAB
     • System statistics
     • Cache performance chart
     • Database information
     • Auto-refresh every 5s

┌──────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                               │
└──────────────────────────────────────────────────────────────────────────────┘

  ❌ "ModuleNotFoundError: No module named 'fastapi'"
     → pip install -r requirements.txt

  ❌ "Address already in use"
     → lsof -ti:8000 | xargs kill -9

  ❌ "Vector database is empty"
     → Add PDFs to pdfs/ folder and restart, OR
     → Go to Ingest tab, upload PDFs, start ingestion

  ❌ "Could not connect to Ollama"
     → ollama serve
     → ollama pull nomic-embed-text
     → ollama pull deepseek-r1:7b
